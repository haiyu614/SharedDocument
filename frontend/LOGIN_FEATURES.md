# 登录/注册界面功能说明

## 新功能概览

已升级的登录页面现在包含完整的注册功能，采用现代化的UI设计。

## 主要特性

### 🎨 视觉设计
- **渐变背景**: 紫色渐变背景配合动态网格动画
- **卡片式布局**: 圆角白色卡片，带有阴影效果
- **响应式设计**: 适配移动端和桌面端
- **流畅动画**: 页面加载、标签切换、按钮交互均有平滑过渡

### 🔐 登录功能
- 用户名和密码输入
- 表单验证
- 加载状态指示器（旋转动画）
- 错误提示信息
- 自动跳转到主页

### ✨ 注册功能
- 用户名输入（最少3个字符）
- 邮箱输入（可选）
- 密码输入（最少6个字符）
- 确认密码输入
- 密码匹配验证
- 注册成功后自动登录
- 成功/错误提示信息

### 🎯 用户体验
- **标签切换**: 登录/注册之间平滑切换
- **图标增强**: 每个输入框都有对应的图标
- **即时反馈**: 输入框聚焦时的视觉反馈
- **自动完成**: 支持浏览器自动填充
- **加载状态**: 提交时显示加载动画
- **错误处理**: 友好的错误提示信息

## 技术实现

### 组件结构
- **LoginPage.vue**: 主组件，包含登录和注册表单
- **Auth Store**: 状态管理，包含 `login()` 和 `register()` 方法
- **Auth Service**: API 调用服务

### 样式特点
- 使用 CSS 变量实现主题化
- Flexbox 布局
- CSS 动画和过渡效果
- 响应式媒体查询

### 表单验证
- 前端验证：密码长度、密码匹配
- 后端验证：通过 API 返回的错误信息
- 实时错误提示

## 使用方法

### 启动开发服务器
```bash
cd frontend
npm install
npm run dev
```

### 访问登录页面
打开浏览器访问: `http://localhost:5173/login`

### 注册新用户
1. 点击"注册"标签
2. 填写用户名（至少3个字符）
3. 填写邮箱（可选）
4. 输入密码（至少6个字符）
5. 确认密码
6. 点击"注册"按钮
7. 注册成功后会自动登录并跳转

### 登录
1. 在"登录"标签下
2. 输入用户名和密码
3. 点击"登录"按钮
4. 登录成功后跳转到主页

## API 接口

### 注册接口
- **端点**: `POST /auth/register`
- **请求体**:
```json
{
  "username": "string",
  "password": "string",
  "email": "string (optional)"
}
```

### 登录接口
- **端点**: `POST /auth/login`
- **请求体**: `application/x-www-form-urlencoded`
- **参数**: `username`, `password`
